# How to View Data Flow Diagrams (DFD)

This guide explains how to view the Data Flow Diagrams for the CodeWave project.

---

## Quick Start (Fastest Method)

### Option 1: View in GitHub
1. **Push `DFD_DIAGRAMS.md` to GitHub**
2. **Open the file on GitHub**
3. **Mermaid diagrams render automatically!**

### Option 2: View in VS Code
1. **Install "Markdown Preview Mermaid Support" extension**
2. **Open `DFD_DIAGRAMS.md`**
3. **Press `Ctrl+Shift+V` (Windows/Linux) or `Cmd+Shift+V` (Mac)**
4. **Diagrams render in preview**

### Option 3: Online Mermaid Editor
1. **Go to https://mermaid.live/**
2. **Copy Mermaid code from `DFD_DIAGRAMS.md`**
3. **Paste and view**

---

## Files Available

- **DFD_DIAGRAMS.md** - Complete DFD documentation with Mermaid diagrams
- **DFD_DIAGRAM_PLANTUML.puml** - PlantUML format for advanced tools

---

## DFD Levels Included

1. **Context Diagram (Level 0)** - System as single process with external entities
2. **Level 1 DFD** - Major processes and data stores
3. **Level 2 DFDs** - Detailed process decompositions:
   - 1.0 Authenticate User
   - 2.0 Manage Assessment
   - 3.0 Manage Learning Content
   - 4.0 Execute Code
   - 5.0 Track Progress
   - 6.0 Manage Quizzes
   - 7.0 Manage Admin Functions
   - 8.0 Generate Reports

---

## Viewing PlantUML Version

1. **VS Code**: Install "PlantUML" extension, open `.puml` file, press `Alt+D`
2. **Online**: Go to http://www.plantuml.com/plantuml/uml/ and paste content
