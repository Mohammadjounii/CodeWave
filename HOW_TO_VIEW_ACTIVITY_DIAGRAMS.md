# How to View Activity Diagrams

This guide explains how to view the Activity Diagrams for the CodeWave project.

---

## Quick Start (Fastest Method)

### Option 1: View in GitHub
1. **Push `ACTIVITY_DIAGRAMS.md` to GitHub**
2. **Open the file on GitHub**
3. **Mermaid diagrams render automatically!**

### Option 2: View in VS Code
1. **Install "Markdown Preview Mermaid Support" extension**
2. **Open `ACTIVITY_DIAGRAMS.md`**
3. **Press `Ctrl+Shift+V` (Windows/Linux) or `Cmd+Shift+V` (Mac)**
4. **Diagrams render in preview**

### Option 3: Online Mermaid Editor
1. **Go to https://mermaid.live/**
2. **Copy Mermaid code from `ACTIVITY_DIAGRAMS.md`**
3. **Paste and view**

## Files Available

- **ACTIVITY_DIAGRAMS.md** - Complete activity diagram documentation with Mermaid diagrams
- **ACTIVITY_DIAGRAM_PLANTUML.puml** - PlantUML format for advanced tools

## Viewing PlantUML Version

1. **VS Code**: Install "PlantUML" extension, open `.puml` file, press `Alt+D`
2. **Online**: Go to http://www.plantuml.com/plantuml/uml/ and paste content

---

## Diagram Features

- **Decision Points**: Yellow diamonds showing conditional flows
- **Start/End Nodes**: Green start, pink end
- **Parallel Processing**: Shown in AD8 (Progress Tracking)
- **Loops**: Repeat-while patterns for iterations
- **Swimlanes**: Can be added for actor-specific flows
