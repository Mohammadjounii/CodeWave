# LinkedIn Post - CodeWave (Short Version)

---

🚀 **Excited to share my latest project: CodeWave!**

So I made CodeWave - it's not an actual code wave in coding, but rather a "code breeze" that made me explore different .NET concepts and build a comprehensive learning platform similar to CodeAcademy! 💻

**What is CodeWave?**
A full-featured Learning Management System (LMS) for teaching Python & Java through interactive courses, hands-on coding exercises, and real-time code execution.

**Key Features:**
✨ Interactive coding exercises with Docker-based code execution
✨ Focus Mode with AI-powered chatbot helper (Google Gemini API)
✨ Assessment system with personalized learning paths
✨ Quiz system and progress tracking
✨ Admin dashboard with analytics and charts
✨ Job offer listings and CV generation

**.NET Concepts I Explored:**

🏗️ **Architecture:**
- Clean Architecture (Domain, Application, Infrastructure, Web layers)
- Repository Pattern & Unit of Work Pattern
- Dependency Injection
- Service Layer pattern

🔐 **Security:**
- ASP.NET Core Identity
- OAuth (Google & GitHub)
- Role-based access control
- Claims-based authorization

💾 **Database:**
- Entity Framework Core (Code-First)
- Database migrations
- Multi-database support (MySQL & SQL Server)
- Complex LINQ queries

🌐 **Web Development:**
- ASP.NET Core MVC 9.0
- Razor Views with Tailwind CSS
- Chart.js for data visualization
- HttpClientFactory for API calls
- Session management

🐳 **Infrastructure:**
- Docker-based code execution
- Database seeding
- Configuration management

This project was an incredible learning journey that deepened my understanding of enterprise-level .NET development and building scalable web applications!
